# Milyen alkalmazások kellenek az Angular-hoz?
Alkalmazás és csomagok is szükségesek az Angular-hoz. Nézzük meg, hogy mik is kellenek.

## Node.js
Node.js is kell nekünk, ha elszeretnénk kezdeni frontend-el foglalkozni. Ez a csomag kb. mindenféle framework-hoz. Például Vue, Angular, React, Ember (igen, ez külön framework), Svelte, etc. Érdemes [itt](https://nodejs.org/en) letölteni, és azonbelül az LTS (Long-Time-Support)-t letölteni. Amint települt, inditsuk újra a gépünket. Majd ezzel a parancsokkal ellenőrizhetjük, hogy ezzel készen vagyunk.
> node -v és npm -v

## Visual Studio Code / VSCodium
Mindkettő ugyanaz a felület, csak a kiadás, ami különbözik. Visual Studio Code ([VSCode](https://code.visualstudio.com/)) zárt forráskódú, miközben [VSCodium](https://vscodium.com/) nyílt. Amint feltelepült, én a vscode-icons extension-t javaslom feltelepíteni.
