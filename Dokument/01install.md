# Hogyan is lehet telepíteni az Angular-t?
Előző fejezetben minden bizonyára feltelepítettük a programokat. Mostani részben az Angular-t fogjuk telepíteni.
## Angular telepítése
Az Angular-t is telepíteni kell. Így nyissuk meg a command line interface-t (cmd.exe), majd ezt a sort írjuk be:
> npm install -g @angular/cli
